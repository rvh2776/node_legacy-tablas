# App creada en curso de: node_legacy.
 
App simple que genera tablas de multiplicar.


```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base de la tabla a crear                    [number] [required]
  -l, --listar   Muestra en consola la tabla creada   [boolean] [default: false]
  -h, --hasta    Hasta que numero debe llegar la tabla    [number] [default: 10]
```

Clonar e instalar

```
npm install

node app.js
```
